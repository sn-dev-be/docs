## ContentType

### **Brief Description**

- Explanation of the message types supported in the `contentType` field when sending messages by the APP administrator, as well as detailed descriptions of the `content` fields for each message type.

### **ContentType Message Type Descriptions**

| ContentType Value | Type Description      |
| ---------------- | -------------------- |
|      101         | Text Message         |
|      102         | Image Message        |
|      103         | Audio Message        |
|      104         | Video Message        |
|      105         | File Message         |
|      110         | Custom Message       |
|      1400        | System Notification  |

### **Content Details**

- The `content` field contains a specific json object, and different message types have different json objects.

#### **Text Message**

```json
{
  ...,
  "content": {
      "content": "hello"
  },
  ....
}
```

| Parameter | Required |  Type  | Description       |
| --------- | -------- | ------ | ----------------- |
|  content     |  Yes     | string | Detailed text content |

#### **Custom Message**

```json
{
  ...,
  "content": {
      "data": "",
      "description": "",
      "extension": ""
  },
  ....
}
```

|   Parameter   | Required |    Type     | Description                                                               |
| ------------ | -------- | ----------- | ------------------------------------------------------------------------- |
|    data      |  Yes     | json string | User-customized message converted from a json object to a string          |
| description  |  No      | json string | Extended description converted from a json object to a string, optional    |
|  extension   |  No      | json string | Extended field, currently unused                                           |

#### **System Notification Message**

> System notifications are a special type of custom notification. The client will create a separate session with a fixed session type of 4, where the sender ID must be one of the admin IDs configured in the backend config file.

```json
{
  ...,
  "content": {
      ...
  },
  ....
}
```

|       Parameter       | Required |  Type  | Description                                                                                                                                                           |
| --------------------- | -------- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  notificationName      |  Yes     | string | Notification title                                                                                                                                                   |
| notificationFaceURL    |  No      | string | Notification avatar URL                                                                                                                                              |
|  notificationType      |  Yes     | int    | Notification type, e.g., 1 for onboarding notifications, 2 for offboarding notifications, etc.                                                                       |
|        text            |  Yes     | string | Notification main content                                                                                                                                            |
|     externalUrl        |  No      | string | The URL to redirect to when the notification is clicked (if left blank, no redirection occurs)                                                                       |
|       mixType          |  Yes     | int    | Notification mixed type<br/>0: Text only<br/>1: Text + Image<br/>2: Text + Video<br/>3: Text + File<br/>4: Text + Audio<br/>5: Text + Audio + Image<br/>             |
|     pictureElem        |  No      | object | Image element                                                                                                                                                        |
|    sourcePicture       |  No      | object | Original picture                                                                                                                                                     |
|     bigPicture         |  No      | object | Large picture                                                                                                                                                        |
|   snapshotPicture      |  No      | object | Thumbnail                                                                                                                                                            |
|      soundElem         |  No      | object | Audio element                                                                                                                                                        |
|      videoElem         |  No      | object | Video element                                                                                                                                                        |
|      fileElem          |  No      | object | File element                                                                                                                                                         |
|         ex             |  No      | string | Extended field                                                                                                                                                       |
