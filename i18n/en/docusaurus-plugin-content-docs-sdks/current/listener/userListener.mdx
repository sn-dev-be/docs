# UserListener

## Module Overview

:::info

User information listener, callbacks are triggered when user information changes.

:::

| Module Name                                                   | Module Description                         |
| ------------------------------------------------------------- | ------------------------------------------ |
| [onSelfUserInfoUpdate](/callback/onSelfUserInfoUpdate.mdx)   | Personal information change notification   |
| [onUserStatusChanged](/callback/onUserStatusChanged.mdx)     | Subscribed user online status change notification |
