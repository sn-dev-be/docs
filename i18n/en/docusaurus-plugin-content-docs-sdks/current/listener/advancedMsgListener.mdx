# AdvancedMsgListener

## Module Overview

:::info

Message listener that is triggered upon receiving new messages, read receipts, and revoked messages.

:::

| Module Name                                                               | Module Description              |
| ------------------------------------------------------------------------- | ------------------------------- |
| [onRecvNewMessage](/callback/onRecvNewMessage.mdx)                        | Receive new message             |
| [onRecvOfflineNewMessage](/callback/onRecvOfflineNewMessage.mdx)          | Receive offline message         |
| [onRecvC2CReadReceipt](/callback/onRecvC2CReadReceipt.mdx)                | Read receipt for one-on-one chat |
| [onRecvGroupReadReceipt](/callback/onRecvGroupReadReceipt.mdx)            | Read receipt for group chat      |
| [onNewRecvMessageRevoked](/callback/onNewRecvMessageRevoked.mdx)          | Callback for revoked message    |
