# ConversationListener

## Module Overview

:::info

Conversation listener, triggered when the conversation changes and the conversation sync status changes.

:::

| Module Name                                                                                | Module Description                           |
| ------------------------------------------------------------------------------------------ | -------------------------------------------- |
| [onSyncServerStart](/callback/onSyncServerStart.mdx)                                       | Synchronization with server session started  |
| [onSyncServerFinish](/callback/onSyncServerFinish.mdx)                                     | Synchronization with server session finished |
| [onSyncServerFailed](/callback/onSyncServerFailed.mdx)                                     | Synchronization with server session failed   |
| [onNewConversation](/callback/onNewConversation.mdx)                                       | New conversation available                  |
| [onConversationChanged](/callback/onConversationChanged.mdx)                               | Key information of certain conversations changed |
| [onTotalUnreadMessageCountChanged](/callback/onTotalUnreadMessageCountChanged.mdx)         | Notification of total unread conversation count change |
