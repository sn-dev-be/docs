# onFriendApplicationAccepted

## Feature Introduction

:::info

When a friend application is accepted, both the applicant and the recipient will receive this callback, and they successfully establish a friendship.

:::

### Call Example

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnFriendApplicationAccepted, ({ data }) => {
  // data - Friend Application
});
```
