# onBlackAdded

## Feature Introduction

:::info

You will receive this callback when a user's blacklist is added.

:::

<TabItem value="Web">

### Return Prototype

```ts showLineNumbers

onBlackAdded(data: WSEvent<BlackUserItem>): void;

```

### Return Results

| Name | Type                                                                                           | Description   |
| ---- | ---------------------------------------------------------------------------------------------- | ------------- |
| data | [WSEvent](/class/response.mdx)<[BlackUserItem](/class/friend/blackInfo.mdx)> | Blacklist information |

### Call Example

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnBlackAdded, ({ data }) => {
  // data is the blacklist information
});
```
