# deleteMessageFromLocalStorage

## Feature Introduction

:::info Note

Delete a message from local storage. After uninstalling the APP, it can be retrieved again.

:::

:::caution Attention

**Relevant Callback**:
[onConversationChanged](/callback/onConversationChanged.mdx)
If the message being deleted is the latest one, the latest message in the conversation will change.

:::
