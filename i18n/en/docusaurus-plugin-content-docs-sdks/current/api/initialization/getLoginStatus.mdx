# getLoginStatus

## Feature Introduction

:::info Description

Retrieve user login status. If this interface is called without initializing the SDK, it will return -1001 (Uninitialized).

:::

:::caution Note

Note the difference between connection status and login status. Only call `login` when not logged in.

:::
