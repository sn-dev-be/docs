# deleteFriend

## Feature Description

:::info Note

Delete a friend.

:::

:::caution Warning

(1) Due to the mutual nature of friendships, this function only removes the other party from one's own friend list;
(2) Deleting a friend does not automatically delete the conversation. To delete the conversation, you can call the function to delete the conversation.

**Relevant callback**:
[onFriendDeleted](../../callback/onFriendDeleted)
:::
