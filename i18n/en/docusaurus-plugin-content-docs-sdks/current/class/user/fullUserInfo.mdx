# FullUserInfo

## Feature Introduction

:::info

Integrates PublicInfo, FriendInfo, and BlackInfo.

When using getuserinfo,
If it's a friend, both publicInfo and friendInfo will be set.
If it's in the blacklist, both publicinfo and blackInfo will be set.
If it's both a friend and in the blacklist, then publicInfo, friendInfo, and blackInfo will all be set.

When using getfriend,
If it's in the blacklist, then both friendInfo and blackInfo will be set, but publicInfo won't be set.

:::
