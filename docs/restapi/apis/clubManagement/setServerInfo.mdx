<center>

## 修改部落信息

</center>

### 简要描述

- 修改部落信息。

### 请求方式

- `post`

### 请求 URL

- `http://x.x.x.x:10002/club/set_server_info`

### Header

| header 名   | 示例值        | 选填 | 类型   | 说明                         |
| :---------- | :------------ | :--- | ------ | ---------------------------- |
| operationID | 1646445464564 | 必填 | string | operationID 用于全局链路追踪 |
| token       | eyJhbxxxx3Xs  | 必填 | string | 管理员 token                 |

### 请求参数示例

```json
{
  "serverInfoForSet": {
    "serverID": "xadxwr24",
    "serverName": "new name",
    "icon": "new icon",
    "description": "new description",
    "banner": "www.newfaceURL.com/xx.png",
    "ex": "new ex",
    "applyMode": 1,
    "searchable": 1,
    "userMutualAccessible": 1
  }
}
```

| 字段名                                | 选填 | 类型   | 说明                                                                                                                                  |
| :------------------------------------ | :--- | :----- | ------------------------------------------------------------------------------------------------------------------------------------- |
| serverInfoForSet                      | 必填 | object | 部落信息对象                                                                                                                          |
| serverInfoForSet.serverID             | 必填 | string | 要修改的部落 ID                                                                                                                       |
| serverInfoForSet.serverName           | 选填 | string | 新部落名                                                                                                                              |
| serverInfoForSet.icon                 | 选填 | string | 新部落头像                                                                                                                            |
| serverInfoForSet.description          | 选填 | string | 新部落介绍                                                                                                                            |
| serverInfoForSet.banner               | 选填 | string | 新部落封面图                                                                                                                            |
| serverInfoForSet.ex                   | 选填 | string | 新部落扩展字段                                                                                                                        |
| serverInfoForSet.applyMode            | 选填 | int    | 进部落是否需要验证；0：所有人进部落需要验证，1：直接进部落 |
| serverInfoForSet.searchable           | 选填 | int    | 是否能搜索落；0：不允许搜索，1：允许搜索                                                        |
| serverInfoForSet.userMutualAccessible | 选填 | int    | 是否能查看用户信息；0：不允许查看部落成员信息，1：允许查看                                                                    |

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```

### 成功返回示例的参数说明

| 参数名  | 类型   | 说明                         |
| :------ | :----- | :--------------------------- |
| errCode | int    | 错误码,0 表示成功            |
| errMsg  | string | 错误简要信息,无错误时为空    |
| errDlt  | errDlt | 错误详细信息,无错误时为空    |
| data    | object | 通用数据对象，具体结构见下方 |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
