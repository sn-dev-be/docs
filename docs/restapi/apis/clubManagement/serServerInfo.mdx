<center>

## 编辑部落

</center>

### 简要描述
- 编辑部落，需要修改哪一项，body中就传哪一项
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/club/set_servers_info` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
    "serverInfoForSet":{
        "serverID": "2054384799",
        "serverName": "2023-11-15",
        "icon": "Gordon",
        "description": "description",
        "banner": "banner",
        "applyMode": 1,
        "searchable": 1,
        "userMutualAccessible": 1,
        "dappID": "123456",
        "ex":"ex"
    }
}

```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|serverID|必填|string|部落ID|
|serverName|选填|string|部落名称|
|icon|选填|string|部落头像|
|description|选填|string|部落介绍|
|banner|选填|string|部落banner|
|applyMode|选填|number|申请模式 0-审核 1-不审核|
|searchable|选填|number|是否可搜索 0-否 1-是|
|userMutualAccessible|选填|number|成员相互访问 0-否 1-是|
|ex|选填|string|部落ID|

### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {}
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|



### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
