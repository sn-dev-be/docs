## 获取部落禁言名单

</center>

### 简要描述

- 获取部落禁言名单
- `post`

### 请求 URL

- `http://x.x.x.x:10002/club/get_server_mute_records`

### Header

| header 名   | 示例值        | 选填 | 类型   | 说明                         |
| :---------- | :------------ | :--- | ------ | ---------------------------- |
| operationID | 1646445464564 | 必填 | string | operationID 用于全局链路追踪 |
| token       | eyJhbxxxx3Xs  | 必填 | string | 管理员 token                 |

### 请求参数示例

```json
{
  "serverID": "499038287",
  "pagination": {
    "showNumber": 10,
    "pageNumber": 1
  }
}
```

| 字段名     | 选填 | 类型   | 说明     |
| :--------- | :--- | :----- | -------- |
| serverID   | 必填 | string | 部落 ID  |
| pagination | 必填 | object | 分页对象 |
| showNumber | 必填 | number | 每页行数 |
| pageNumber | 必填 | number | 当前页码 |

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "total": 1,
    "records": [
      {
        "muteRecord": {
          "serverID": "499038287",
          "blockUserID": "10010",
          "createTime": 1700106602485,
          "operatorUserID": "65386e3a57fdf1e72485393f",
          "muteEndTime": 1700106602485
        },
        "operatorUser": {
          "userID": "",
          "nickname": "",
          "faceURL": ""
        },
        "blockUser": {
          "userID": "",
          "nickname": "",
          "faceURL": ""
        }
      }
    ]
  }
}
```

### 成功返回示例的参数说明

| 参数名         | 类型   | 说明                         |
| :------------- | :----- | :--------------------------- |
| errCode        | int    | 错误码,0 表示成功            |
| errMsg         | string | 错误简要信息,无错误时为空    |
| errDlt         | errDlt | 错误详细信息,无错误时为空    |
| data           | object | 通用数据对象，具体结构见下方 |
| total          | number | 总记录数                     |
| records        | Array  | 禁言列表                     |
| muteRecord     | Object | 禁言基本数据                 |
| blockUserID    | string | 被禁言用户 ID                |
| operatorUserID | string | 操作用户 ID                  |
| serverID       | string | 部落 ID                      |
| createTime     | number | 操作时间                     |
| muteEndTime    | number | 禁言结束时间                 |
| operatorUser   | Object | 操作用户                     |
| userID         | string | 用户 ID                      |
| nickname       | string | 昵称                         |
| faceURL        | string | 头像                         |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
