## 获取部落房间指定成员

</center>

### 简要描述

- 获取部落房间指定成员的详细信息
- `post`

### 请求 URL

- `http://x.x.x.x:10002/club/get_server_group_members_info`

### Header

| header 名   | 示例值        | 选填 | 类型   | 说明                         |
| :---------- | :------------ | :--- | ------ | ---------------------------- |
| operationID | 1646445464564 | 必填 | string | operationID 用于全局链路追踪 |
| token       | eyJhbxxxx3Xs  | 必填 | string | 管理员 token                 |

### 请求参数示例

```json
{
  "groupID": "499038287",
  "userIDs": ["", ""]
}
```

| 字段名   | 选填 | 类型   | 说明         |
| :------- | :--- | :----- | ------------ |
| groupID | 必填 | string | 部落房间 ID      |
| userIDs  | 必填 | Array  | 用户 ID 集合 |

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "total": 2,
    "members": [
      {
        "serverID": "873584727",
        "userID": "65386e3a57fdf1e72485393f",
        "roleLevel": 100,
        "joinTime": 1700222957428,
        "nickname": "Dada123",
        "faceURL": "https://s3.amazonaws.com/mimoios4726d2326a5b4e83895a1221d766c82c221922-dev/avatar/1699858612889.069_65386e3a57fdf1e72485393f",
        "serverRoleID": "2561981564",
        "inviterUserID": "65386e3a57fdf1e72485393f",
        "muteEndTime": 86400000
      }
    ]
  }
}
```

### 成功返回示例的参数说明

| 参数名       | 类型   | 说明                         |
| :----------- | :----- | :--------------------------- |
| errCode      | int    | 错误码,0 表示成功            |
| errMsg       | string | 错误简要信息,无错误时为空    |
| errDlt       | errDlt | 错误详细信息,无错误时为空    |
| data         | object | 通用数据对象，具体结构见下方 |
| total        | number | 总记录数                     |
| members      | Array  | 部落成员列表                 |
| serverID     | string | 部落 ID                      |
| userID       | string | 用户 ID                      |
| nickname     | string | 昵称                         |
| faceURL      | string | 头像                         |
| serverRoleID | string | 角色 ID                      |
| muteEndTime  | number | 禁言时间                     |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
