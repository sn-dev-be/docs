## 获取部落详情

</center>

### 简要描述
- 根据给定serverID获取部落详情
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/club/get_servers_info` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
  "serverIDs": ["3165603661", "1082887520"]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|serverIDs|必填|string|部落ID|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|server|object|部落基本信息|
|server.serverID|string|部落ID|
|server.serverName|string|部落名|
|server.icon|string|部落logo|
|server.banner|string|部落banner|
|server.ownerUserID|string|部落主ID|
|server.memberNumber|number|成员数量|
|server.categoryNumber|number|分组数量|
|server.channelNumber|number|房间数量|
|server.dappID|string|部落推荐dapp|
|categoryList|Array|分组列表|
|categoryList.categoryID|string|分组ID|
|categoryList.categoryName|string|分组名称|
|categoryList.groupList|Array|房间列表|
|categoryList.groupList.groupID|string|房间ID|
|categoryList.groupList.groupName|string|房间名称|
|categoryList.groupList.groupType|number|房间类型 - IM用|
|categoryList.groupList.conversationID|string|会话ID|
|categoryList.groupList.conversationType|number|会话类型|
|categoryList.groupList.GroupCategoryID|string|分组ID|
|categoryList.groupList.ServerID|string|部落ID|
|categoryList.groupList.GroupMode|number|房间类型 1-聊天房间 2-应用房间|
|categoryList.groupList.dapp|object|应用数据|
|categoryList.groupList.dapp.dappID|string|dappID|
|joined|bool|是否已加入该房间，缺省值为false|


### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
