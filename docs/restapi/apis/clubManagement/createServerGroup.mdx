## 创建房间

</center>

### 简要描述
- 创建房间
- `post` 
### 请求URL
- `http://x.x.x.x:10002/club/create_server_group` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
  "groupInfo":{
    "groupName": "2023-11-15",
    "faceURL": "2023-11-15",
    "introduction": "introduction",
    "ownerUserID": "payAdmin",
    "groupType":3,
    "groupMode": 2,
    "conditionType": 0,
    "condition": 0,
    "groupCategoryID":"2149539263",
    "serverID":"2054384799"
  },
  "dappID": "dappID"
}

```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupInfo|必填|Object|房间基本信息|
|faceURL|必填|string|房间logo|
|introduction|选填|string|房间介绍|
|ownerUserID|必填|string|房主ID|
|groupName|必填||string|房间名称|
|groupType|number|房间类型 部落房间类型为3|
|groupCategoryID|string|分组ID|
|serverID|string|部落ID|
|groupMode|number|房间类型 1-聊天房间 2-应用房间|
|dappID|选填|string|dappID|
|conditionType|number|token 类型 0-无要求 1-ETMP|
|condition|number|token数量，默认为0|

### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {
      "groupInfo":{
        "groupID": "groupID",
        "groupName": "2023-11-15",
        "faceURL": "2023-11-15",
        "introduction": "introduction",
        "ownerUserID": "payAdmin",
        "groupType":3,
        "groupMode": 2,
        "conditionType": 0,
        "condition": 0,
        "groupCategoryID":"2149539263",
        "serverID":"2054384799"
      }
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|groupInfo|object|groupInfo|
|grouoID|string|房间ID|
|faceURL|string|房间logo|
|introduction|string|房间介绍|
|ownerUserID|string|房主ID|
|groupName|string|房间名称|
|groupType|number|房间类型 部落房间类型为3|
|groupCategoryID|string|分组ID|
|serverID|string|部落ID|
|groupMode|number|房间类型 1-聊天房间 2-应用房间|
|dappID|选填|string|dappID|
|conditionType|number|token 类型 0-无要求 1-ETMP|
|condition|number|token数量，默认为0|

### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
