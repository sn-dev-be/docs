## 获取热门部落列表

</center>

### 简要描述
- 获取热门部落列表, 缺省值为默认值时，接口响应字段为空
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/club/get_server_recommended_list` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {
        "servers": [
            {
                "serverInfo": {
                    "serverID": "1736658435",
                    "serverName": "2023-11-15",
                    "icon": "Gordon",
                    "banner":"",
                    "description": "description",
                    "ownerUserID": "payAdmin",
                    "memberNumber": 1,
                    "categoryNumber": 3,
                    "channelNumber": 4,
                    "createTime": 1700014726950
                },
                "memberAvatarList": [
                    "baidu.com"
                ]
            }
        ]
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|servers|Array|部落集合|
|serverInfo|object|部落基本信息|
|serverInfo.serverID|string|部落ID|
|serverInfo.serverName|string|部落名|
|serverInfo.icon|string|部落logo|
|serverInfo.ownerUserID|string|部落主ID|
|serverInfo.memberNumber|number|成员数量|
|serverInfo.categoryNumber|number|分组数量|
|serverInfo.channelNumber|number|房间数量|
|serverInfo.banner|string|部落背景|
|memberAvatarList|Array|用户头像列表|


### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
