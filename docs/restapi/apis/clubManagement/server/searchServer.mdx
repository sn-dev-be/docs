## 搜索部落

</center>

### 简要描述
- 搜索部落
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/club/search_server` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
  "keyword": "499038287",
  "pagination":{
      "showNumber":10,
      "pageNumber":1
  }
}


```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|keyword|必填|string|关键字|
|pagination|必填|object|分页对象|
|showNumber|必填|number|每页行数|
|pageNumber|必填|number|当前页码|

### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {
        "total": 3,
        "serverInfos": [
            {
                "serverID": "2054384799",
                "serverName": "2023-11-15",
                "icon": "Gordon",
                "description": "description",
                "ownerUserID": "payAdmin",
                "memberNumber": 1,
                "status": 1,
                "categoryNumber": 3,
                "channelNumber": 4,
                "createTime": 1700014726950,
                "dappID": "123456"
            }
        ]
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空| 
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|number|总记录数|
|serverInfos|Array|部落列表|
|serverID|string|部落ID|
|serverName|string|部落名|
|icon|string|部落logo|
|banner|string|部落banner|
|ownerUserID|string|部落主ID|
|memberNumber|number|成员数量|
|categoryNumber|number|分组数量|
|channelNumber|number|房间数量|
|dappID|string|部落推荐dapp|



### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
