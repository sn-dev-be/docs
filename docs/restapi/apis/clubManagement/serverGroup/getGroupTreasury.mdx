<center>

## 获取房间金库

</center>

### 简要描述

- 获取应用房间金库配置
- `post`

### 请求 URL

- `http://x.x.x.x:10002/club/get_group_treasury`

### Header

| header 名   | 示例值        | 选填 | 类型   | 说明                         |
| :---------- | :------------ | :--- | ------ | ---------------------------- |
| operationID | 1646445464564 | 必填 | string | operationID 用于全局链路追踪 |
| token       | eyJhbxxxx3Xs  | 必填 | string | 管理员 token                 |

### 请求参数示例

```json
{
  "groupIDs": ["3821772881"]
}
```

| 字段名   | 选填 | 类型   | 说明    |
| :------- | :--- | :----- | ------- |
| groupIDs | 必填 | string | 房间 ID |

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "records": [
      {
        "groupID": "3821772881",
        "treasuryID": "0ab4b25a-1bae-4c69-a6b6-8b4365b23067",
        "icon": "abc.png",
        "name": "alg-13",
        "walletType": 7,
        "contractAddress": "0ab4b25a-1bae-4c69-a6b6-8b4365b23067",
        "administratorAddress": "112346789"
      }
    ]
  }
}
```

### 成功返回示例的参数说明

| 参数名               | 类型   | 说明                      |
| :------------------- | :----- | :------------------------ |
| errCode              | int    | 错误码,0 表示成功         |
| errMsg               | string | 错误简要信息,无错误时为空 |
| errDlt               | errDlt | 错误详细信息,无错误时为空 |
| treasuryID           | string | 金库 ID                   |
| groupID              | string | 房间 ID                   |
| icon                 | string | 金库 ICON                 |
| name                 | string | 金库名称                  |
| walletType           | number | 钱包链类型                |
| contractAddress      | string | 金库合约地址              |
| administratorAddress | string | 管理员合约地址            |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
