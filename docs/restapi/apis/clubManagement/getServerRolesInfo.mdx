## 获取部落指定部落角色信息

</center>

### 简要描述
- 获取群成员列表
- `post` 
### 请求URL
- `http://x.x.x.x:10002/club/get_server_roles_info` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|
|token|eyJhbxxxx3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
  "roleIDs":["awe","wewe"]
}

```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|roleIDs|必填|Array|角色ID集合|

### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {
        "roles": [
            {
                "serverID": "499038287",
                "roleID": "6535cf2d2558948bad220ce5",
                "roleName": "Didi123",
                "colorLevel": 100,
                "ex":"",
                "createTime": 1700103910020,
                "priority": 0,
                "icon": "https://avatars.githubusercontent.com/u/28507035?s=200&v=4",
                "permissions": "{\"sendMsg\":true,\"PostTweet\":true,\"manageMsg\":true,\"TweetReply\":true,\"manageRole\":true,\"shareServer\":true,\"manageMember\":true,\"manageServer\":true,\"manageChannel\":true,\"manageCommunity\":true,\"ManageGroupCategory\":true}"
            }
        ]
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|roles|Array|部落角色列表|
|serverID|string|部落ID|
|roleID|string|角色ID|
|roleName|string|角色名|
|colorLevel|number|颜色等级|
|ex|string|备注|
|createTime|int64|创建时间|
|priority|number|优先级|
|icon|string|角色头像|
|permissions|string|角色权限|

### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": ": [1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
