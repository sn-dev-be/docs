## ContentType

### **简要描述**

- 管理员消息发送字段中 contentType 支持的消息类型说明以及消息 content 的具体字段说明。

### **ContentType 消息类型说明**

| ContentType 值 | 类型说明     |
| :------------: | :----------- |
|      2002      | 转账成功通知 |
|      2003      | 转账失败通知 |
|      2004      | 充值成功通知 |
|      2005      | 充值失败通知 |
|      2006      | 提现成功通知 |
|      2007      | 提现失败通知 |
|      2008      | 红包超时通知 |
|      2009      | 收到转账通知 |
|      2010      | 钱包绑定解除通知 |

### **Content 具体内容**

- content 通知内容字符串，必须是对应 json 的字符串

#### **支付相关**

```json
{
  "sendID": "openIMAdmin",
  "sendNickname": "openIMAdmin",
  "recvID": "65129c018cf7320debfac1af",
  "recvNickname": "65129c018cf7320debfac1af",
  "coinNum": "500",
  "coinType": "ETH",
  "content": "充值详情",
  "transactionStatus": 1,
  "transactionType": 1,
  "transactionHash": "0xabc"
}
```

|      参数名       | 必选 |  类型  | 说明                   |
| :---------------: | :--: | :----: | :--------------------- |
|      sendID       |  是  | string | 发送者 ID              |
|   sendNickname    |  是  | string | 发送者昵称             |
|      recvID       |  是  | string | 接收者 ID              |
|   recvNickname    |  是  | string | 接收者昵称             |
|      coinNum      |  是  | string | 货币数量               |
|     coinType      |  是  | string | 货币类型               |
|      content      |  是  | string | 支付说明               |
| transactionStatus |  是  |  int   | 支付状态 1-成功 2-失败 |
|  transactionType  |  是  |  int   | 支付类型 1-钱包 2-红包 |
|  transactionHash  |  是  | string | 支付订单 hash          |
