# onServerDismissed

## 功能介绍

:::info

部落被解散时，该部落所有成员会收到此回调。

:::

### 调用示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnServerDismissed, ({ data }) => {
  // data 群信息
});
```
