# OnServerUnreadMessageCountChanged

## 功能介绍

:::info

部落所有未设置免打扰的会话总未读发生变化时的回调。

:::

### 返回原型


### 调用示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnServerUnreadMessageCountChanged, ({ data }) => {
  // data 消息未读数
});
```
