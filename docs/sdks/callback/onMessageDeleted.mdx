# onMsgDeleted

## 功能介绍

:::info

接收到消息被删除时会收到此回掉，回调中只会携带一条消息。

:::

### 调用示例

```js showLineNumbers

import { CbEvents } from 'open-im-sdk-wasm/constant';

IMSDK.on(CbEvents.OnMsgDeleted, ({ data }) => {
  // data 新消息
});
```
