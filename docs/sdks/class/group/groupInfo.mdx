### GroupInfo

| 字段名称               | 字段类型                                                  | 描述                                      |
| ---------------------- | --------------------------------------------------------- | ----------------------------------------- |
| groupID                | String                                                    | 群唯一 ID                                 |
| groupName              | String                                                    | 群名称                                    |
| notification           | String                                                    | 群公告                                    |
| introduction           | String                                                    | 群简介                                    |
| faceURL                | String                                                    | 群头像链接                                |
| createTime             | int                                                       | 群创建时间                                |
| status                 | int                                                       | 群状态 0 正常，1 被封，2 解散，3 禁言     |
| creatorUserID          | String                                                    | 创建者 userID                             |
| groupType              | [GroupType](docs/sdks/enum/groupType.mdx)                 | 群状态                                    |
| ownerUserID            | String                                                    | 群主 userID                               |
| memberCount            | int                                                       | 群成员数量                                |
| ex                     | String                                                    | 应用层扩展字段                            |
| needVerification       | [GroupVerification](docs/sdks/enum/groupVerification.mdx) | 进群验证方式                              |
| lookMemberInfo         | int                                                       | 不允许通过群获取成员资料 0：关闭，1：打开 |
| applyMemberFriend      | int                                                       | 不允许通过群添加好友 0：关闭，1：打开     |
| notificationUpdateTime | int                                                       | 群公告更新时间                            |
| notificationUserID     | String                                                    | 群公告发布者 userID                       |

</TabItem>
