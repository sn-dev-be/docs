# ServerInfo

## 功能介绍

:::info

部落信息。

:::

## 部落信息{#ServerInfo}

| 字段名称             | 字段类型 | 描述                                     |
| -------------------- | -------- | ---------------------------------------- |
| serverID             | string   | 部落 ID                                  |
| serverName           | string   | 部落名称                                 |
| banner               | string   | 部落封面图                               |
| description          | string   | 部落简介                                 |
| icon                 | string   | 部落头像                                 |
| createTime           | int64    | 申请进部落时间                           |
| status               | int32    | 部落状态：0 正常，1 被封，2 解散，3 禁言 |
| creatorUserID        | string   | 建部落者 ID                              |
| ownerUserID          | string   | 部落主 ID                                |
| memberNumber         | uint32   | 部落成员数量                             |
| searchable           | int32    | 是否可被搜到： 0 能，1 不能              |
| userMutualAccessible | int32    | 是否可以双向访问成员 ： 0 能，1 不能     |
| ex                   | int32    |                                          |
| dappID               | int32    | dappID                                   |
