### ClubApplicationInfo

| 字段名称      | 字段类型                               | 描述                                             |
| ------------- | -------------------------------------- | ------------------------------------------------ |
| serverID      | String                                 | 部落 ID                                          |
| serverName    | String                                 | 部落名称                                         |
| banner        | String                                 | 部落封面图                                       |
| description   | String                                 | 部落简介                                         |
| icon          | String                                 | 部落头像                                         |
| createTime    | int                                    | 申请进部落时间                                   |
| status        | int                                    | 部落状态：0 正常，1 被封，2 解散，3 禁言         |
| creatorUserID | String                                 | 建部落者 ID                                      |
| ownerUserID   | String                                 | 部落主 ID                                        |
| memberNum     | int                                    | 部落成员数量                                     |
| userID        | String                                 | 进部落申请者 ID                                  |
| nickname      | String                                 | 进部落申请者昵称                                 |
| userFaceURL   | String                                 | 进部落申请者头像                                 |
| handleResult  | [int](docs/sdks/enum/handleResult.mdx) | 处理结果                                         |
| reqMsg        | String                                 | 申请加部落的附加信息                             |
| handledMsg    | String                                 | 处理附加信息                                     |
| reqTime       | int                                    | 申请进部落时间                                   |
| handleUserID  | String                                 | 处理进部落的用户 ID                              |
| handledTime   | int                                    | 处理近期申请时间                                 |
| ex            | String                                 | 扩展字段                                         |
| joinSource    | [int](docs/sdks/enum/joinSource.mdx)   | 进部落方式 2：通过邀请 3：通过搜索 4：通过二维码 |
| inviterUserID | String                                 | 邀请进部落用户 ID                                |

</TabItem>
