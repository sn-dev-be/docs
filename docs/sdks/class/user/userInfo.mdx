### UserInfo

| 字段名称         | 字段类型 | 描述                                                               |
| ---------------- | -------- | ------------------------------------------------------------------ |
| userID           | String   | 用户 ID                                                            |
| nickName         | String   | 用户昵称                                                           |
| faceURL          | String   | 用户头像                                                           |
| ex               | String   | 用户扩展字段                                                       |
| createTime       | int      | 用户注册时间（毫秒）                                               |
| appMangerLevel   | int      | 内部字段，可忽略                                                   |
| attachedInfo     | String   | SDK 内部扩展字段，应用层无需关注                                   |
| globalRecvMsgOpt | int      | 全局免打扰 0：正常；1：不接受消息；2：接受在线消息不接受离线消息； |
| allowBeep        | int      | 新消息提醒声音 1-开启 2-关闭                                       |
| allowVibration   | int      | 新消息提醒震动 1-开启 2-关闭                                       |
| allowPushContent | int      | 新消息提醒聊天内容 1-开启 2-关闭                                   |
| allowOnlinePush  | int      | 新消息应用内推送 1-开启 2-关闭                                     |

    AllowPushContent int32  `gorm:"column:allow_push_content" json:""`
    AllowOnlinePush  int32  `gorm:"column:allow_online_push" json:""`
