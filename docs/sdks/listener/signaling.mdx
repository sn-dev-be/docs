# SignalingListener

:::info

操作信令监听器，当收到邀请、拒绝、取消等信令时回调

:::

| 模块名称                                           | 模块功能简介       |
| -------------------------------------------------- | ------------------ |
| [OnReceiveNewInvitation](docs/sdks/callback/.mdx)  | 邀请信令           |
| [OnInviteeAccepted](docs/sdks/callback/.mdx)       | 邀请接受信令       |
| [OnInviteeRejected](docs/sdks/callback/.mdx)       | 邀请拒绝信令       |
| [OnJoined](docs/sdks/callback/.mdx)                | 加入房间信令       |
| [OnInvitationCancelled](docs/sdks/callback/.mdx)   | 邀请取消信令       |
| [OnHangUp](docs/sdks/callback/.mdx)                | 挂断信令           |
| [OnClosed](docs/sdks/callback/.mdx)                | 关闭信令           |
| [OnMicphoneStatusChanged](docs/sdks/callback/.mdx) | 麦克风状态改变信令 |
| [OnSpeakStatusChanged](docs/sdks/callback/.mdx)    | 说话状态通知信令   |
