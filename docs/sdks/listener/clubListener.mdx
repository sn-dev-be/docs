# ClubListener

## 模块概览

:::info

部落监听器，当部落及部落成员信息改变时回调。

:::

| 模块名称                                                                      | 模块功能简介       |
| ----------------------------------------------------------------------------- | ------------------ |
| [onClubMemberAdded](docs/sdks/callback/onClubMemberAdded.mdx)                 | 有新成员加入部落   |
| [onClubMemberDeleted](docs/sdks/callback/onClubMemberDeleted.mdx)             | 有成员离开部落     |
| [onClubMemberInfoChanged](docs/sdks/callback/onClubMemberInfoChanged.mdx)     | 某成员信息发生变更 |
| [onJoinedClubAdded](docs/sdks/callback/onJoinedClubAdded.mdx)                 | 部落成员变更       |
| [onJoinedClubDeleted](docs/sdks/callback/onJoinedClubDeleted.mdx)             | 部落成员变更       |
| [onClubInfoChanged](docs/sdks/callback/onClubInfoChanged.mdx)                 | 部落信息变更       |
| [onClubApplicationAccepted](docs/sdks/callback/onClubApplicationAccepted.mdx) | 部落申请被接受通知 |
| [onClubApplicationAdded](docs/sdks/callback/onClubApplicationAdded.mdx)       | 新的部落申请通知   |
| [onClubApplicationDeleted](docs/sdks/callback/onClubApplicationDeleted.mdx)   | 部落申请删除通知   |
| [onClubApplicationRejected](docs/sdks/callback/onClubApplicationRejected.mdx) | 部落申请被拒绝通知 |
| [onClubDismissed](docs/sdks/callback/onClubDismissed.mdx)                     | 部落解散通知       |
