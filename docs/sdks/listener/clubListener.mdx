# ClubListener

## 模块概览

:::info

部落监听器，当部落及部落成员信息改变时回调。

:::

| 模块名称                                                                               | 模块功能简介       |
| -------------------------------------------------------------------------------------- | ------------------ |
| [onServerMemberAdded](docs/sdks/callback/club/onServerMemberAdded.mdx)                 | 有新成员加入部落   |
| [onServerMemberDeleted](docs/sdks/callback/club/onServerMemberDeleted.mdx)             | 有成员离开部落     |
| [onServerMemberInfoChanged](docs/sdks/callback/club/onServerMemberInfoChanged.mdx)     | 某成员信息发生变更 |
| [onJoinedServerAdded](docs/sdks/callback/club/onJoinedServerAdded.mdx)                 | 部落成员变更       |
| [onJoinedServerDeleted](docs/sdks/callback/club/onJoinedServerDeleted.mdx)             | 部落成员变更       |
| [onServerInfoChanged](docs/sdks/callback/club/onServerInfoChanged.mdx)                 | 部落信息变更       |
| [onServerApplicationAccepted](docs/sdks/callback/club/onServerApplicationAccepted.mdx) | 部落申请被接受通知 |
| [onServerApplicationAdded](docs/sdks/callback/club/onServerApplicationAdded.mdx)       | 新的部落申请通知   |
| [onServerApplicationDeleted](docs/sdks/callback/club/onServerApplicationDeleted.mdx)   | 部落申请删除通知   |
| [onServerApplicationRejected](docs/sdks/callback/club/onServerApplicationRejected.mdx) | 部落申请被拒绝通知 |
| [onServerDismissed](/sdks/callback/club/onServerDismissed)                             | 部落解散通知（可用）       |
| [onServerMemberKicked](/sdks/callback/club/onServerMemberKicked)                       | 部落成员被踢出通知（可用） |
| [OnServerGroupDismissed](/sdks/callback/club/onServerGroupDismissed)                       | 部落房间解散通知（可用） |
