# AdvancedMsgListener

## 模块概览

:::info

消息监听器，当收到新消息、已读回执及撤回消息时回调

:::

| 模块名称                                                                            | 模块功能简介                |
| ----------------------------------------------------------------------------------- | --------------------------- |
| [onRecvNewMessage](docs/sdks/callback/onRecvNewMessage.mdx)                         | 收到新消息                  |
| [onRecvOfflineNewMessage](docs/sdks/callback/onRecvOfflineNewMessage.mdx)           | 收到离线消息                |
| [onRecvC2CReadReceipt](docs/sdks/callback/onRecvC2CReadReceipt.mdx)                 | 单聊消息已读回执            |
| [onRecvGroupReadReceipt](docs/sdks/callback/onRecvGroupReadReceipt.mdx)             | 群聊消息已读回执            |
| [onNewRecvMessageRevoked](docs/sdks/callback/onNewRecvMessageRevoked.mdx)           | 撤回消息回调                |
| [onRecvRedPacketStatusChanged](docs/sdks/callback/onRecvRedPacketStatusChanged.mdx) | 红包状态改变回调            |
| [onRecvGroupVoiceCallJoined](docs/sdks/callback/.mdx)                               | 群聊语音房用户加入回调      |
| [onRecvSelfSendRedPacketSeqChanged](docs/sdks/callback/.mdx)                        | 用户发送红包后 seq 更新回调 |
| [onMsgDeleted](docs/sdks/callback/onMessageDeleted.mdx)                             | 删除消息回调                |
