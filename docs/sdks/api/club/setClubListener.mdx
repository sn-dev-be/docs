# setClubListener

## 功能介绍

:::info 说明

设置部落事件监听器，对于部落变化的相关事件，进行异步回调通知，以便 UI 能及时感知并处理。

:::

:::caution 注意

（1）在 initSDK 之后立刻调用；  
（2）只能调用一次。

:::
