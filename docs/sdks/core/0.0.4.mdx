## 获取对话框是否允许发送消息

### 功能介绍

:::info 说明
获取对话框是否允许发送消息

:::

### 函数原型

```go showLineNumbers
func AllowedSendMsg(callback open_im_sdk_callback.Base, operationID string, targetUserID string)
```

### 输入参数

| 参数名称       | 参数类型 | 是否必填 | 描述    |
| -------------- | -------- | -------- | ------- |
| targetUserID | string   | 是       | 目标用户 ID json 数组 |

### 返回结果

| 名称 | 类型 | 描述                                                              |
| ---- | ---- | ----------------------------------------------------------------- |
| ~    | int  | 1-可以 0-不可以 |

### 代码示例

```go showLineNumbers

```
